# dependency-injection-cat

Dependency Injection Cat is a TypeScript-only library implement the Dependency Inversion pattern with Dependency Injection

## Installation
Install with yarn

```bash
yarn add di-cat
```

Install with npm

```bash
npm install di-cat
```
